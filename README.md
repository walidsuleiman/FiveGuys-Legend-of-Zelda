# This is Group FiveGuys' 3902 Legend of Zelda Project

**Legend of Zelda**\
W to move forward\
A to move left\
S to move back\
D to move right\
1 to equip wood sword\
2 to equip white sword\
3 to unequip swords\
N to use attack 1\
M to use attack 2\
E to take damage and die\
R to Reset\
B to mute\
Q to Quit\
Z to use boomerang\
C to open inventory\
Enter to Pause / Menu
Use the doors to go to another room
